# Book-Assistant  

RAG Book Assistant is an AI-powered document question-answering application built using Streamlit, LangChain, ChromaDB, Hugging Face Embeddings, and Mistral AI. Users can upload PDF documents, which are processed into text chunks and converted into vector embeddings. The application retrieves the most relevant document sections using semantic search and generates accurate responses based only on the retrieved context. This improves answer quality while minimizing hallucinations, providing an efficient and user-friendly way to interact with document content.



# Objectives :- 

1. To develop an AI-powered application for answering questions from PDF documents.
2. To implement Retrieval-Augmented Generation (RAG) for accurate and context-aware responses.
3. To use Hugging Face embeddings and ChromaDB for efficient document retrieval.
4. To integrate the Mistral language model for generating relevant answers based on retrieved content.
5. To provide a simple and interactive user interface for document upload and question answering using Streamlit.



# Key Features :- 

1. Upload PDF documents.
2. Extract text from PDF files.
3. Split documents into manageable chunks.
4. Generate embeddings using Hugging Face models.
5. Store document embeddings in ChromaDB.
6. Retrieve relevant document chunks using MMR retrieval.
7. Generate answers using the Mistral LLM.
8. Modern Streamlit user interface.
9. Context-aware question answering.
10. Fast semantic search over uploaded documents.



# Result :- 

1. Successfully built a Retrieval-Augmented Generation (RAG) application for PDF documents.
2. Generated accurate answers using retrieved document context.
3. Improved response quality by combining semantic search with Mistral AI.
4. Stored embeddings efficiently in ChromaDB for fast retrieval.
5. Delivered an interactive and user-friendly interface using Streamlit.



# Conclusion :-

The RAG Book Assistant successfully combines document retrieval and large language models to provide accurate, context-aware answers from uploaded PDF documents. By using Hugging Face embeddings, ChromaDB, and Mistral AI, the system offers efficient semantic search and reliable responses while reducing hallucinations. The project demonstrates the practical application of Retrieval-Augmented Generation for intelligent document analysis and question answering.

4. Embedding Model: Hugging Face
